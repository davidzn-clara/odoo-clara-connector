# Clara Connector for Odoo 17

This is the designated official interface linking Odoo to the Clara Spend Management ecosystem (v3 API). It securely maps and fetches transactions applying mTLS schemas without requiring on-disk certificates on the deployment environment.

## Installation

1. Clone or download this module into your Odoo `addons` path.
2. Ensure Odoo dependencies `hr_expense` and `account` are installed.
3. Update the App List and install **Clara – Spend Management Integration**.

## Configuration & First Sync Checklist

Follow these steps exactly to execute your first sync:
- **Set Up API Credentials**: Navigate to Settings -> Accounting -> Clara Connector. Select your Region and insert the Client ID and Secret generated by Clara.
- **Upload Certificates**: Upload `cert.pem`, `ca.pem`, and `key.pem` directly in the backend UI. They will be encrypted sequentially into the database.
- **Test Connection**: Press the `Test Connection` button. A green badge indicates success.
- **Map your Accounting**: Assign typical Odoo accounts to handle expenses, Clara Corporate liability, default journals, and products.
- **Sync Now**: Navigate to the Clara application on the top bar. Press `Sync Now` to manually fetch the transactions via the pop-up Wizard. You can view progress logs under the **Sync Logs** menu.
